# 일자리 박람회 구직자-기업 자동 매칭 MVP

내부 확인용 테스트 웹앱입니다. 실제 운영용이 아닙니다.

## 실행

```bash
npm install
npm run dev -- --host 0.0.0.0
```

## 테스트/빌드

```bash
npm test
npm run build
```

## 테스트 관리자 계정

- admin1@example.com / test1234
- admin2@example.com / test1234

## 내부 공유 안내 문구

아래 문구를 카카오톡이나 이메일에 함께 보내세요.

```text
[일자리 박람회 매칭 웹앱 MVP 테스트 안내]

아래 주소는 내부 확인용 테스트 웹앱입니다.
실제 운영 서비스가 아니며, 기능·화면·업무흐름 확인 목적입니다.

테스트 주소:
{여기에 테스트 주소 입력}

주의사항
- 실제 개인정보를 입력하지 마세요.
- 실제 이력서를 업로드하지 마세요.
- 실제 기업정보를 입력하지 마세요.
- 테스트용 가짜 정보만 입력해 주세요.
- 이메일은 실제 발송되지 않고, 관리자 화면의 이메일 로그로만 시뮬레이션됩니다.
- 관리자 계정 정보는 검토자에게만 개별 공유해 주세요.

테스트 관리자 계정
- admin1@example.com / test1234
- admin2@example.com / test1234
```

## 구현된 기능

- 구직자 신청 폼
- 개인정보 수집·이용·제공 동의서 스크롤 박스 및 체크박스
- 이력서 파일 확장자/10MB 제한 검증
- 업종 기준 자동 매칭
- 기업 담당자/매칭 담당자 이메일 발송 로그 시뮬레이션
- 관리자 로그인
- 실시간 대시보드: 총 신청, 매칭, 미매칭, 업종별 신청 수
- 구직자 목록/상세 조회
- 고용기업 직접 등록
- 테스트 구직자 10명, 고용기업 10개 기본 데이터

## 제한사항

- 실제 이메일 발송 없음
- 실제 서버 DB 없음: 브라우저 localStorage 사용
- CSV 업로드는 MVP 화면 설명만 포함, 운영 버전에서 구현 필요
- 실제 개인정보 처리 전 보안/법무 검토 필요

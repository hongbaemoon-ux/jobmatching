# 일자리 박람회 이력서 사전 접수 MVP

내부 확인용 테스트 웹앱입니다. 실제 운영용이 아닙니다.

## 실행

```bash
npm install
npm run dev -- --host 0.0.0.0
```

## 테스트/빌드

```bash
npm test
npm run build
```

## 테스트 관리자 계정

- admin1@example.com / test1234
- admin2@example.com / test1234

## 외부 홈페이지 연동 가정

외부 홈페이지에서 기업 모집공고의 **이력서 사전 접수** 버튼을 누르면 아래처럼 `postingId` 또는 `companyId`를 붙여 이 웹앱으로 이동합니다.

```text
http://웹앱주소/?postingId=c1
http://웹앱주소/?companyId=c1
```

웹앱은 해당 기업을 자동 체크하고, 같은 직무의 다른 채용공고를 함께 보여줍니다. 공고 전문은 웹앱에 저장하지 않고 **원본 채용공고 보기** 링크로 외부 홈페이지 새 창을 엽니다.

## 구현된 기능

- 외부 홈페이지에서 선택한 공고 자동 체크 흐름
- 같은 직무의 다른 채용공고 목록 표시
- 공고 목록에 기업명, 모집부문, 직무, 근무지역(시·군·구), 요약, 원본 공고 링크 표시
- 구직자가 체크한 기업 담당자에게 이력서 이메일 발송 로그 시뮬레이션
- 개인정보 수집·이용·제공 동의서 문안 교체 및 스크롤 박스/필수 체크박스
- 이력서 파일 확장자/10MB 제한 검증
- 관리자 로그인
- 6개 탭 관리자 대시보드
  - 핵심현황
  - 직무별 현황
  - 기업별 현황
  - 구직자별 상세
  - 이메일 발송 로그
  - 원본 공고 연결 관리
- 자동 선택 기업 해제 가능 및 안내문 표시
- 구직자 목록/상세 조회
- 고용기업 직접 등록
- 테스트 구직자 50명, 고용기업 50개 기본 데이터

## 제한사항

- 실제 이메일 발송 없음
- 실제 서버 DB 없음: 브라우저 localStorage 사용
- 원본 공고 URL은 현재 테스트용 `example.com` 링크
- CSV 업로드는 MVP 화면 설명만 포함, 운영 버전에서 구현 필요
- 실제 개인정보 처리 전 보안/법무 검토 필요
